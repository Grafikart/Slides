CSS Sémantique

Avantages

  • Découple le fond de la forme (ex: CSS Garden)
  • HTML plus lisible

Inconvénients

  • Trouver des noms de classes
  • Risque de duplication
  • Maintenance difficile si manque de rigueur

Avantages

  • Itération plus rapide
  • Meilleure expérience de développement

Inconvénients

  • Risque d'un manque de cohérence
  • Peu de mutualisation
  • Nécessite un outillage (CSS à la demande)
  • On se prive de l'imbrication

Exemples

  • FrontEnd : React, VueJS, Svelte, Solid, Angular...
  • Backend : include ou fonction
  • Python 
    {% app_button "Supprimer" variant="danger" %}
  • PHP : Laravel : Blade component, Symfony : Twig component 
    <x-button variant="primary">Acheter</x-button>
  • Ruby on Rails : View component ou erb 
    <%= render(Button.new(variant: "primary", text: "Acheter")) %>
