Gestion de l'état

React, Preact

Utilisation d'un setter


          const [count, setCount] = useState(3);

          const increment = () => {
            setCount(v => v + 1);
          }
          const reset = () => {
            setCount(0);
          }

Angular, Preact, Solid, VueJS

Utilisation de signaux / observable


          const count = signal(0);

          const increment = () => {
            count.update(v => v + 1)
          }
          const reset = () => {
            count.set(0)
          }
          console.log("La valeur : " + count())
